例题 1：P3367 【模板】并查集

题目：P3367 【模板】并查集

思路

模板题，考虑用一个数组 faifa_ifai​ 记录 iii 的负责人，函数 find(i)find(i)find(i) 查询 iii 的族长。

于是可以写

int find(int x){
    if(fa[x]==x) return x;//当查询到负责人时返回负责人
    return fa[x];
}

但这样貌似会超时。

于是需要路径压缩，如下图：

也就是说，在访问一个点的族长时把所有访问到的结点的负责人直接设为族长，像这样：

int find(int x){
    if(f[x]==x) return x;
    return f[x]=find(f[x]);
}

代码

//P3367
#include <bits/stdc++.h>
using namespace std;
int f[10010];
int find(int a){
    if(f[a]==a) return a;
    return f[a]=find(f[a]);
}
int main(){
    int n,m;
    cin>>n>>m;
    int x,y,z;
    for(int i=0;i<=n;i++) f[i]=i;
    for(int i=0;i<m;i++){
        cin>>z>>x>>y;
        if(z==1){
            f[find(x)]=find(y);
        }
        else{
            if(find(x)==find(y)) cout<<'Y'<<endl;
            else cout<<'N'<<endl;
        }
    }
    return 0;
}
